## What You Can Do Now

### 1. **Visit the Application**
Open your browser and go to: **http://localhost:3000**

### 2. **Sign Up Options**

You'll see two options:
- **Sign In** - For existing users
- **Sign Up** - To create a new account

### 3. **Try the Application**

#### Sign Up as First Responder:
1. Click "Sign Up"
2. Choose "Sign Up as Responder"
3. Fill in your details:
   - Personal information
   - Vehicle details
   - Certifications (BLS, CPR AED, ACLS, PALS)
4. You'll be redirected to the Responder Dashboard

#### Sign Up as 911 Operator:
1. Click "Sign Up"
2. Choose "Sign Up as Operator"
3. Fill in your details:
   - Name, email, password
   - Badge number
4. You'll be redirected to the Operator Dashboard

### 4. **What You'll See**

#### First Responder Dashboard:
- Emergency request cards
- Filter by urgency (Critical, High, Moderate)
- Filter by case type
- Sort by distance
- Click on emergencies to see details with maps
- Interactive Mapbox maps showing locations

#### Operator Dashboard:
- Create new emergency calls
- View all active emergencies
- See statistics (active calls, critical cases, etc.)
- Monitor emergency status
- Dispatch and manage calls

### 5. **Test Accounts**

Since this is using localStorage for auth, you can:
- Create any email/password you want
- Sign up multiple times to test different roles
- Data persists in your browser's localStorage
